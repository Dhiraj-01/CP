### Problem
```
Given an positive integer X and string S, where '0' <= S[i] <= '9'
Calculate the minimum number of deletions required in S, to make it divisible.
int(S) % X == 0

Constraints:
len(S) <= 10000, X <= 100

Input:
S = "123456", 
X = 13

Output:
2

Explanation:
By deleting 4 and 6 from the string
S = "1235"
Now S % X = 0
So ans in this case is 2
```

### Solution
```
Hint: knapsack varient ?
at each index two choice either keep it or delete it

dp[index][modulo] => min cost to reach given modulo % x in string s with (0, index) char
```

<details>
    <summary>code c++</summary>

```cpp
ll x;
string s;
ll dp[10000][100];

ll f(ll index, ll mod)
{
    dbg(index, mod);
    if(index >= s.size()) {
        return mod == 0 ? 0 : 1e9;
    }
    ll& ans = dp[index][mod];
    if(ans != -1) return ans;
    ans = min(
        1 + f(index + 1, mod),
        f(index + 1, (mod * 10 + (s[index] - '0')) % x)
    );
    return ans;
}

void solve()
{
    x = 13;
    s = "123456";
    memset(dp, -1, sizeof(dp));
    cout << f(0, 0) << endl;    
}
```
